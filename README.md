# SMILE Lab Website

Static website for the Soni Microfluidics & Interfacial Engineering Laboratory (SMILE Lab), E. S. Witchger School of Engineering, Marian University.

## Files

- `index.html` — Home
- `research.html` — Research
- `publications.html` — Publications
- `people.html` — People
- `opportunities.html` — Opportunities
- `contact.html` — Contact Us
- `assets/css/style.css` — Design system and layout
- `assets/images/` — Website images

## Notes

The Project 2 image on the Research page is currently a placeholder. Replace `assets/images/project-microfluidics-placeholder.png` with the representative image from the Lab on a Chip paper and update the image filename in `research.html` if needed.

## Local Preview

Open `index.html` in a browser, or run a local server from this folder:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
